1. Inline Element:- It is used to provide an inline element to the webpage. Like :- a, span, b, i, u, em, strong, dfn, code, samp, kbd, var, cite, abbr, acronym, q, sub, sup.
  2. Block Element:- It is used to provide block element to the webpage. Like:- section,p, article, nav, aside, footer, header, main, h1, h2, h3, h4, h5, h6
