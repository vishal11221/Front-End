1. What is HTML?
    • HTML is the standard markup language for creating web pages.
  2. What are sementic and non sementic tags?
    • Sementic tags are those tag that have some meaning.for example:- header, nav, main, section, article, aside, footer.
    • Non sementic tags are those tag that have no meaning.for example:- div, span, p, h1, h2, h3, h4, h5, h6
  3. What are inline and block elements?
    • Inline elements are those element that align in single line not break the line. for example span
    • Block elements are those element that break the line for example div, paragraph.
  4. What are the feature added in Html5 ?
    • Audio,Video,Placeholder,Header,footer,Regular Expression.
  5. Difference between div and section ?
    • div is non sementic tag while section is sementic tag
    • section represents a completely different section of the web page while div just represents the tag inside it.
  6. What happen, when Doctype not given in HTML ?
    • If we not declare doctype then many features not supported like:- header, nav, main, section, article, aside, footer and also give validation error.
  7. Why we are using method attribute using form?
    • Method attribute specify to the web browser how to send form data to a server.
  8. Difference between Post and Get method?
    • Post method is used to send data to the server while get method is used to retrieve data from the server
    • In Get method form data is shown in URL while in Post method form data is not shown in URL
  9. What are the attribute present in form ?
    • action, method, name, target, autocomplete.
  10. What is the difference between label and legend tag ?
    • Label tag is used to label the input field while legend tag is used to label the form.
  11. What are self closing tags ?
    • Self closing tags are those tag that can be close or not close no effect. for example:- input, select, textarea, button, label.
  12. What is the need of action attribute and for and id in form ?
    • action attribute is used to submit the form data to the database.
    • for attribute is used to specifies which form element a label is bound to.
    • id attribute is used to specify a unique id for an HTML Element.
  13. What is character encoding?
    • Character encoding is a method of converting bytes into characters.
    • UTF-8 is a character encoding standard.
    • To validate or display an html document properly, a program must choose a proper character encoding.
