1. Property Binding:- It is one of the core features used for binding data from the component class to the HTML template. It is denoted by [].
  2. Property binding lets you display component class properties in the template (HTML) using square brackets [ ].
    • app.component.ts
    • Example:- name = "Angular";
    • app.component.html
    • Example:-

      [name]

    • Example:-

      [name | uppercase]
