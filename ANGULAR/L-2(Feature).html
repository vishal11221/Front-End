1. Component-Based Architecture:- Angular apps are built using components, which are reusable and modular blocks of UI with their own logic and templates.
  2. Data Binding:- Angular supports data binding, which allows you to bind data from the model to the view.
  3. Two-Way Data Binding:- Angular supports two-way data binding, which allows you to bind data from the view to the model and vice versa.
  4. Template Driven Forms:- Angular provides template-driven forms, which allow you to create forms without writing any code.
  5. Dependency Injection:- Angular uses dependency injection to manage dependencies between components and services.
  6. Routing:- Angular provides routing capabilities to navigate between different views and components.
  7. Directives:- Angular provides structural and attribute directives to manipulate the DOM, like *ngIf, *ngFor, and custom ones.
  8. Forms Support:- Angular provides built-in support for forms, including form validation and data binding.
  9. HTTPClient Module:- Angular provides an HTTP client for making HTTP requests, including GET, POST, PUT, and DELETE methods.
  10. Angular CLI:- Angular CLI is a command-line interface for creating and managing Angular projects.
