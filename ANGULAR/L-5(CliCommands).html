1. ng new project-name:- It is used to create a new Angular project.
  2. ng generate component component-name:- It is used to generate a new component.
  3. ng generate service service-name:- It is used to generate a new service.
  4. ng generate module module-name:- It is used to generate a new module.
  5. ng add @angular/material:- It is used to add Material Design to the Angular project.
  6. ng serve:- It is used to run the Angular project.
  7. ng build:- It is used to build the Angular project.
