1. Before installing Angular, you need to install Node.js & npm:- Angular requires Node.js (which comes with npm - Node Package Manager).
    • node -v
    • npm -v
  2. Step 1:- Install Angular cli:- npm install -g @angular/cli, check it version as:- ng version
  3. Step 2:- Create Angular project:- ng new project-name
  4. Step 3:- Navigate to the project:- cd project-name
  5. Step 4:- Run the project:- ng serve
