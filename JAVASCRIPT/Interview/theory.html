1. Javascript is a Client Side Scripting Language. It is used to create dynamic and interactive web pages.
  2. Javascript is cross-platform language. It can be used in any platform.
  3. Datatype in Javascript:- String, Number, Boolean, Null, Undefined
  4. Difference between Null and Undefined:- Null is a value, Undefined is a variable. For example:- When we declare a variable but not assign any value to it then it is undefined. When we assign null to it then it is null.

  6. Difference between var, let and const:- var can be updated and re-declared , Is function-scoped, not block-scoped. let can be updated but not re-declared , Is block-scoped. const cannot be updated or re-declared , Is block-scoped.
  7. Difference between Function Scope and Block Scope ? Function declare with their function name and the variable inside a function declare it scope or access within a function not outside of function that declare with var keyword. Block have no name, it declare with curly braces and the variable inside a block declare it scope or access within a block not outside of block that declare with let or const keyword.
  8. Local & Global Scope:- Local scope is a variable that is declared inside a function or within a block that have some boundaries. Global scope is a variable that is declared outside a function or outside a block that have no boundaries, user can access it anywhere.
  9. Difference between == and === :- == checks only value. === checks value and datatype. for example:- var x = 5; var y = "5"; console.log(x == y); console.log(x === y);

  14. What is prototype in Javascript:- Prototype is a JavaScript object that is used to add properties and methods to other objects. Ex:- function Person(name, age) { this.name = name; this.age = age; } Person.prototype.sayHello = function() { console.log("Hello, my name is " + this.name + " and I am " + this.age + " years old."); }; var person1 = new Person("John", 30); person1.sayHello(); // Output: Hello, my name is John and I am 30 years old.

What is setTimeout and setInterval in JavaScript:- setTimeout() is a JavaScript function that executes a specified function once after a given delay (in milliseconds). setInterval() is a JavaScript function that repeatedly executes a specified function at fixed time intervals (in milliseconds) until it is stopped using clearInterval(). Ex:- console.log("Start"); setTimeout(function() { console.log("Executed after 2 seconds"); }, 2000); console.log("End"); Ex:- let count = 0; const intervalId = setInterval(function() { count++; console.log("Count:", count); // Stop the interval after it reaches 5 if (count === 5) { clearInterval(intervalId); console.log("Interval stopped"); } }, 1000);

  18. Difference between asynchronous and synchronous:- In synchronous execution, tasks are completed one at a time in a single sequence. The code runs from top to bottom, with each line waiting for the previous one to complete before moving on. Each task blocks the next one until it finishes. In asynchronous execution, tasks are executed concurrently, allowing multiple tasks to run in parallel. // Example of synchronous execution console.log("Task 1"); console.log("Task 2"); console.log("Task 3"); // Example of asynchronous execution console.log("Task 1"); setTimeout(() => { console.log("Task 2"); }, 1000); console.log("Task 3"); // Output: Task 1, Task 3, Task 2 //Note:- In asynchronous code,some tasks like setTimeout,API calls,promises are delayed and other code continues running without waiting for them.

What is callback in JavaScript:- Callback is a function that is passed as an argument to another function and is executed after the completion of that function. // Example of a callback function function greet(name) { console.log(`Hello, ${name}!`); } function sayHello(callback, name) { callback(name); } sayHello(greet, "John"); // Output: Hello, John!

What is IIFE in JavaScript:- IIFE is an Immediately Invoked Function Expression, which is a self-executing function. // Example of an Immediately Invoked Function Expression (IIFE) (function() { console.log("This code is executed immediately"); })(); // function expression var result = (function() { return "This code is executed immediately"; })(); console.log(result);
  21. What is arrow function ? How it differ from regular function:- Arrow function is a function that is defined using the => operator. It is a shorter way to write a function. for example:- var add = (a, b) => a + b; var multiply = (a, b) => a * b; console.log(add(2, 3)); console.log(multiply(2, 3));
