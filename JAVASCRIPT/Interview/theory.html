  20. Difference between asynchronous and synchronous:- In synchronous execution, tasks are completed one at a time in a single sequence. The code runs from top to bottom, with each line waiting for the previous one to complete before moving on. Each task blocks the next one until it finishes. In asynchronous execution, tasks are executed concurrently, allowing multiple tasks to run in parallel. // Example of synchronous execution console.log("Task 1"); console.log("Task 2"); console.log("Task 3"); // Example of asynchronous execution console.log("Task 1"); setTimeout(() => { console.log("Task 2"); }, 1000); console.log("Task 3"); // Output: Task 1, Task 3, Task 2 //Note:- In asynchronous code,some tasks like setTimeout,API calls,promises are delayed and other code continues running without waiting for them.
  21. Difference between async and await ? async and await both are keywords in JavaScript used to handle asynchronous operations. The main difference is that async declare a function as asynchronous and it always returns a Promise, while await is used to pause the execution of an asynchronous function until a Promise is resolved or rejected.
