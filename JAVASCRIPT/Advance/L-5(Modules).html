1. Modules are the combination of group of code like different function with their local and global scope.
  2. JavaScript modules use the **import** and **export** keywords to share code between files.
  1. math.js (module file):- export const add = (a, b) => a + b; export const subtract = (a, b) => a - b;
  2. main.js:- import { add, subtract } from './math.js'; console.log(add(5, 3)); // 8 console.log(subtract(5, 3)); // 2
