Give one example,Switch Statement is different from other conditional operators ?
if-then-else statement can test expressions based on ranges of values or conditions, whereas a switch statement tests expressions based only on a single integer, enumerated value, or String object.
Syntax:-
switch(expression){ case condition 1:statement(s) break; case condition 2:statement(s) break; case condition 3:statement(s) break; default:statement(s) }
