1. ceil(x):-It always return upper value,like if you give 1.5 then it return 2,if you give 2.4 then it return 3.
  2. floor(x):-It always return downward value,like if you give 1.5 then it return 1,if you give 2.4 then it return 2 only.
  3. round(x):-It always round nearest value,like if you give 0.65 then it return 1,if you give 1.4 then it return 1.
  4. trunc(x):-Trunc simply take integer value and return if you give 8.19 then it return 8, it not take after decimal value.
  5. max(x,y....n):-It always return maximum value from following given value.
  6. min(x,y....n):-It always return minimum value from following given value.
  7. sqrt(x):-It return square root of given value.
  8. cbrt(x):-It return cube root of given value.
  9. pow(x,y):-Powerof calculate power of given no,in this you pass two parameter.if you give 4,3 then it return 64 because 4 to the power 3 is 64.
  10. random():-It return random value from your given range value.Like you give 1 to 10 then it return random value from 1 to 10.
  11. abs():-It always return absolute value,if you give positive or negative value.
  12. PI:-It always return 22/7 or 3.14
