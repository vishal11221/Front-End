Location Object Property:-
  1. hash:-The location.hash property sets or returns the anchor part of a URL,including the hash sign(#).
  2. host:-it return host + port also.
  3. hostname:-it return name of your host.
  4. href:-it return url of your website.
  5. origin:-what the origin path of your website.
  6. pathname:-it return the path of your filename
  7. port:-it return the port no of your url.
  8. protocol:-it retun protocol of your website.
  9. search:-it return just after question mark value from your url.
    Location Method:-
  1. Assign():-it assign new url to your current location.
  2. Reload():-it refresh your page.
  3. Replace()

Difference between host and hostname is that host return host + portname but hostname return only its name
