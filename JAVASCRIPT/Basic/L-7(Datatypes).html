Primitive Datatypes:--
  1. String(Value defined in single,double and backticks)
  2. Number(Value defined as integer)
  3. Boolean(true,false)
  4. Null(value defined as null)
  5. Undefined(declare variable but not assigned value init)
    Non Primitive Datatypes:--
  1. Array(Store multiple different value in a single variable,defined with square braces),Array used number indexes.
  2. Object(Store multiple different value in single variable,defined with curley braces),Object used named indexes.
  3. Function:-Function is a block of code that is used to user to reuse the code with function call.
