Number method always return number when you give string or any other value.

parseInt method always return Integer value if you give float value.

parseFloat method always return Float value if you give string value.

isFinite method check value will be countable or not,if value is countable then it return true otherwise false.

isInteger method check value is Integer or not if value is integer then it return true otherwise false.

Note:-If parseInt not find any value from given no then it return NAN.
