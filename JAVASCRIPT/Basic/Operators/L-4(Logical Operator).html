Logical operators are used to test the relationship between two or more Boolean expressions and return a Boolean value based on the result.

    Types of Logical operators:-
  1. AND(&&):-If both condition are true that operator return true otherwise it return false.
  2. OR(||):-If any one condition true that operator retrun true otherwise it return false.
  3. NOT(!):-If value not match then it return true otherwise false.
