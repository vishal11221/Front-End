DOM-Targeting Methods:-
  1. document.getElementById(#demo)
  2. document.getElementByClassName(.demo)
  3. document.getElementByTagName(p)
  1. document:-With the help of document we see our current file document with opening html tag and closing html tag.
  2. document.all:-With the help of document we see the particular document with their index no.
  3. document.documentElement:-
  4. document.head:-It return the head tag,opening head and closing tag.
  5. document.title:-It return the title tag,opening title and closing tag.
  6. document.body:-It return the body tag,opening body and closing body tag.
  7. document.images:-It return all the image tag from your current document.
  8. document.anchors:-It return the all anchor tag from your current document.
  9. document.links:- It return the all links tag from your current document.
  10. document.forms:- It return the all the forms that present in your current document.
  11. document.doctype:- It return doctype of your current document.
  12. document.url:- It return all the url in your document.
  13. document.baseurl:- It same like url,it also return url of your website
  14. document.domain:- It return domain of your website.
  15. document.getElementById(#header):-It target value from particular Id to get the value.
  16. document.getElementByClassName(.header):-It target value from particular class to get the value.
  17. document.getElementByTagName(p):-It target value from particular tag to get the value.
