Difference between oninput and onchange event ?
oninput event happen when we input something in that area but onchange event happen when we loose focus from that area or when we click on outside then it happen.For example we have input field in that we write text,when we write text in input field at a time value print in screen, same as at the time of onchange when we write text in input field then value will not print on the screen at a time when we click on outside input field then value print.
