1. insertAdjacentElement:-That method insert a tag at specific position.At a time it create a tag and insert it where you want.
    1. beforebegin:-It append just before the div start.
    2. afterbegin:-It append just after the div text.
    3. beforeend:-It append just before the div text.
    4. afterend:-It append just after the div end.
  2. insertAdjacentHTML:-That method help to create tag and text in one line and help to append both.
  3. insertAdjacentText

Note:-append method & insertBefore only append that means it work only already created elements.But insertAdjacentElement,insertAdjacentHTML and insertAdjacentText they created and append both.
