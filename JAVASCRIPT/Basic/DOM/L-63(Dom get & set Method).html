DOM Get Method:-
  1. innertText:-It return internal text of selected tag.
  2. innerHTML:-It return internal tag of selected tag.
  3. getAttribute:-It return the attribute value of selected tag.
  4. getAttributeNode:-It return the attribute value with their property.
  5. Attributes:-It return all the attribute from target tag.
    DOM Set Methods:-
  1. innertText:-It set the text in your selected tag.
  2. innerHTML:-It set the tag and text in your selected tag.
  3. setAttribute:-It set the new attribute in your selected tag.
  4. Attribute:-It set the new attribute value in your selected tag by their index no.
  5. removeAttribute:-It remove the attribute from your selected tag.
