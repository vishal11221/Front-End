1. add(class1,class2,.....)
  2. remove(class1,class2,....)
  3. toogle(class)
  4. contains(class)
  5. item(index)
  6. Length

With the help of add method we add new attribute in a particular tag. Like new class in a particular tag.

With the help of remove method we remove new attribute from a particular tag.

With the help of toogle property we add and remove attribute from particular tag.

With the help of contains method we find that particular attribute is available in a particular tag or not.If class exist then it return true otherwise it return false.

With the help of item method we find the attribute from a particular tag with their index no also.

With the help of length property we find the length of particular tag.That means how many attributes are there in a particular tag.
