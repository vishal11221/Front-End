1. keydown:-When we keydown from our keyboard then that event happen.
  2. keypress:-When we keypress from our keyboard then that event happen.
  3. keyup:-When we keyup from our keyboard then that event happen.
  4. focus:-When we mouse hover on field that event apply.
  5. blur:-When we loose the focus then that event apply.
  6. input:-When we type something in that area then that event happen.
  7. change:-When we change value from input field that event trigger.
  8. select:-When we select the value from input field then that event trigger.
  9. submit:-When we submit the form value then that event trigger.
  10. invalid:-invalid event happen when we give click on submit button but the form value is invalid then that event trigger.
