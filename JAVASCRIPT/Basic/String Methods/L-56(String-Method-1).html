length is not a method in javascript,it is a property that find the length of your array and string value,it count space also.

lowercase is string method that is used to show your existing string value into small letter only.

uppercase is string method that is used to show your existing string value into capital letter only.

includes method is string method that is used to find data from your existing string,if data match then it return true otherwise false.it is case sensitive.

startsWith method is string method that is used to find data from first index,if data match then it return true otherwise false.It is also case sensitive.

endsWith method is string method that is use to find data from lastindex,if data match then it return true otherwise false.It is also case sensitive.

search method also works like includes method that means it also find the data from existing string but it return index value.if data is not there then it return -1.

match method is used to find the value from existing string and return new array.We match the value inside double slace.

indexOf method search the value from first index and return the index no if value find successfully.

lastIndexOf method search the value from last index and return the last index no if value find successfully.

replace method is used to replace the existing value and set new value,in this we pass two parameter first is old value and second is new value.

trim method is used to remove extra spaces from your existing string.Basically it is used in html form of input field to remove extra spaces.
