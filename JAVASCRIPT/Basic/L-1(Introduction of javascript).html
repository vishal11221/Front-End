About Javascript:-
  1. JavaScript is client-side scripting language that allows you to make web pages interactive.
  2. JavaScript can change HTML content and structure dynamically without reloading the page using DOM.
  3. JavaScript allows interaction with users via clicks, key presses, mouse movements, etc.
  4. JavaScript can update and change both HTML and CSS with the help of DOM Feature.
  5. JavaScript can calculate, manipulate and validate data.
  6. HTML and CSS are languages that give structure and style to web pages, JavaScript gives web pages interactive elements that engage a user.

Why we use JavaScript not other ?
Unlike other programming languages, JavaScript is known as a client-side language. JavaScript code on a website gets processed on its own using the resources that are available on the system of the user. Therefore, there is no need for any server-side browser to process the code.
