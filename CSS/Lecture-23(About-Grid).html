Grid:- Grid is a two-dimensional layout system with rows and columns.

Difference between Grid and Flexbox:- Flexbox is one dimensional layout to align item horizontally or vertically. Grid is two dimensional layout to align item horizontally or vertically.

Grid property is a shorthand property for grid-template-columns, grid-template-rows, grid-template-areas, grid-auto-rows, grid-auto-columns, grid-auto-flow, grid-template and grid-template-areas.
