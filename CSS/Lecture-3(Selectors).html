1. Id Selector:- It is used to give unique id to the element based on id name user can target to that particular element for styling and so on.It define with # symbol.
  2. Class Selector:- It is used to give different class to the element based on class name user can target to that particular element for styling and so on.It define with . symbol.
  3. Group Selector:- With the help of Group Selector user can target multiple element for styling.It define with , symbol.
  4. Universal Selector:- It is used to target all the element for styling. It define with * symbol.
  5. Element Selector:- It is used to target specific element for styling. It define with element name
