1. What is CSS?
    • CSS is the language we use to style an HTML document.
  2. What css rule set ?
    • A CSS rule set contains one or more selectors and one or more declarations.
    • A selector is a part of a rule set that specifies the element(s) a particular style should be applied to.
  3. What are the css framework are there ?
    • Bootstrap is a popular CSS framework.
    • Tailwind is a popular CSS framework.
  4. What are the css selectors are there ?
    • Id selector:- It is used to give unique id to the element based on id name user can target to that particular element for styling and so on.It define with # symbol.
    • Class selector:- It is used to give different class to the element based on class name user can target to that particular element for styling and so on.It define with . symbol.
    • Element selector:- It is used to target specific element for styling. It define with element name, like p, h1, span.
    • Grouping selector:- With the help of Group Selector user can target multiple element for styling.It define with , symbol.
    • Universal selector:- It is used to target all the element for styling. It define with * symbol.
  5. What is the difference between Pseudo class and Pseudo element ?
    • Pseudo class represents a state of a selector like :hover,:active,:last-child,etc. while pseudo element represents a part of a selector like ::before,::after,etc
  6. What is css flexbox and write all the properties of flexbox?
    • Flexbox is a layout model that is used to align our content at specific position.
      • Left to right:- flex-direction:row
      • Right to left:- flex-direction:row-reverse
      • Top to bottom:- flex-direction:column
      • Bottom to top:- flex-direction:column
  7. When to use css grid and when to use css flex?
    • When we align our content in horizontal as well as vertical like two dimensional, so we use css grid.
    • When we align our content in single axis like one dimensional then we use css flex.
  8. What are the Box-model properties are there ?
    • The css box model is a container that contains multiple properties including borders,margin,padding and the content itself.
  9. Explain css position property ?
    • Static:- Html elements are positioned static by default.Static positioned elements are not effected by the top,bottom,left and right.
    • Relative:-It adjust relative to its original position within the parent element.
    • Absolute:-It is position absouletely with reference to the closest parent that has a position relative value.
    • Fixed:-Fixed is positioned relative to the viewport, which means,it always stays in the same place even,if the page is scrolled.A fixed element does not leave a gap in the page where it would normally have been located.
    • Sticky:-A sticky element toggles between relative & fixed, depending on the scroll position.It is position relative until a given offset position is met in the viewport then it sticks.
  10. Difference between position fixed and sticky ?
    • Fixed:- Scroll with page no, based on viewport, stay visible always , commanly used in floating buttons and navbar.
    • Sticky:- Scroll with page yes until it sticks, based on scrollable container, it stay visible only while its container is in view.Commanly used in sticky navbar.

    • The main reason for using em or % is to allow the user to change the text size without breaking the design.
