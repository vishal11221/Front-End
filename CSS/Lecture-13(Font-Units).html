Font Units:- It is used to set the font size to the particular element.

  1. em:- 10em means 10 multiply of its parent font size. If parent has 2px then child 10em means 20px.
  2. rem:- resize root font element, if root value is 25 and bottom value is 3rem then 3rem means 25*3=75px
  3. vh:- viewport height means 1vh means 1% of viewport height, if user give 100vh that means it capture full screen.
  4. vw:- viewport width means 1vw means 1% of viewport width, if user give 100vw that means it capture 100% width.
